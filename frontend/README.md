# 🧠 Multi-Agent Research System

A full-stack AI research assistant built using:

- ⚡ FastAPI (backend)
- 🤖 Multi-Agent system (LangChain + Mistral)
- 🌐 Web scraping (BeautifulSoup)
- 🎨 React frontend (modern UI)

## 🚀 Features

- 🔍 Search Agent (web research)
- 📄 Reader Agent (scrapes best source)
- ✍️ Writer Agent (generates report)
- 🧠 Critic Agent (reviews quality)
- ⚡ Real-time streaming UI
